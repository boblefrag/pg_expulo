# Columns configuration options

## Name

The name of the column, is mandatory

```code
  "columns": [
     {
       "name": "name",
     }
   ]
```


## Generator

`generator` is the way to control which data will be inserted in the target table, all values are documented in [generator.md]

```code
  "columns": [
     {
       "name": "name",
       "generator": "null"
     }
   ]
```
